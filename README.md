Startup

```
pnpm run dev
```

pnpm install

```
pnpm setup
-> (reload console)
npx pnpm add -g pnpm
```


> branches

```
feature/<...>   | обновление функционала
fix/<...>       | исправление недочетов
```

> commits

```
feat: <...>     | доработка функционала
fix: <...>      | исправление ошибок
refactor: <...> | рефакторинг
```

> merge request

```
(fix / feat): <суть задания либо название первого коммита>
```
